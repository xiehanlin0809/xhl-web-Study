###5.2 循环语句
5.2.1  while循环
while语句是一种前测试语句（先判断以下在执行），循环的目的是为了反复执行语句或代码块，只要表达式为true就继续循环，一旦为假的时候，循环退出

5.2.2  do...while循环
do...while语句是一种后测试语句，就是在循环体中的代码执行之后，才开始判断条件，也就是说，在表达式判断黄子健，循环体内的代码至少会被执行一次

            var a=0;
			do{
				alert('你好');
				a++;
			}while (a<5);

5.2.3  for循环
for语句是一种前测试语句，一般情况下，有三个基础表达式语句，每一个要用分号隔开
          
          //  初始化  判断  更新
          for(var x=0;x<y,x++;){
            //执行循环体
				alert(x);
			}
      //顺序为：1→2→4→3
